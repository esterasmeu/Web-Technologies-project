## Overview

Role-based task management app which stores data in localStorage.

### What it does or has?
- **Login by role:** Admin, Manager, or User flows with preserved credentials.
- **Manage tasks:** Create, assign, update status (OPEN → PENDING → COMPLETION REQUESTED → COMPLETED → CLOSED), set priority, and view history.
- **Review/approve:** Managers approve completion requests; admins can oversee everything.
- **Team chat**
- **Announcements section**
